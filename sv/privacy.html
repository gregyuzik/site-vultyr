Integritetspolicy

Senast uppdaterad: 11 april 2026

Sammanfattning

Vultyr-appen samlar inte in, lagrar eller överför några personuppgifter. Vultyr-webbplatsen (vultyr.app) använder cookielös Google Analytics för att förstå aggregerad besökstrafik. Denna sida förklarar båda i detalj.

Appen — datainsamling

Vultyr-appen samlar inte in någon personlig information. Den kräver inget konto, innehåller inga tredjeparts-SDK:er för analys eller spårning och ringer inte hem till någon server som vi driver.

Appen — nätverksförfrågningar

Appen gör direkta HTTPS-förfrågningar till offentliga statussides-API:er (såsom Statuspage.io, Apple, Google Cloud och andra) för att kontrollera tjänsternas status. Dessa förfrågningar går direkt från din enhet till tjänstens offentliga API — de passerar inte via någon server som vi driver.

Appen — datalagring

All data lagras lokalt på din enhet med Apples SwiftData-ramverk. Om du aktiverar iCloud-synkronisering synkroniseras din lista med bevakade tjänster via Apples iCloud Key-Value Store, som styrs av Apples integritetspolicy. Vi ser aldrig dessa data.

Appen — aviseringar mellan enheter

Om du aktiverar aviseringar mellan enheter delas statusändringar mellan dina enheter via Apples iCloud Key-Value Store. När din Mac upptäcker en statusändring skriver den en lättviktssignal till ditt iCloud-konto. Dina andra enheter observerar ändringen och kör sin egen lokala kontroll. Ingen tredjepartsserver är inblandad — all kommunikation går via Apples iCloud-infrastruktur. Du kan slå på eller av detta från valfri enhet.

Appen — favikoner

Tjänsternas favikoner hämtas från Googles offentliga favikontjänst och cachas lokalt på din enhet.

Webbplatsen — analys

Den här webbplatsen (vultyr.app) använder Google Analytics 4 i cookielöst, IP-anonymiserat läge för att räkna aggregerade sidvisningar. Mer specifikt konfigurerar vi gtag.js med anonymize_ip: true, client_storage: 'none', allow_google_signals: false och allow_ad_personalization_signals: false. Det innebär att ingen _ga-cookie sätts, din IP trunkeras före lagring och inga annonsidentifierare samlas in. Själva Vultyr-appen innehåller ingen analys.

Webbplatsen — tredjepartsdomäner

Att ladda vultyr.app kontaktar följande tredjepartsdomäner:

  • www.googletagmanager.com — laddar gtag.js-skriptet
  • www.google-analytics.com / *.analytics.google.com — tar emot anonymiserade sidvisningssignaler
  • www.google.com/g/collect — tar emot samma anonymiserade sidvisningssignaler (reserv-endpoint för Google Analytics 4)

Vi laddar inte Google Fonts (typsnittet Audiowide är självhostat på vultyr.app) och använder inte en tredjeparts favikontjänst för webbplatsens eget bildmaterial.

Appen — tredjepartstjänster

Vultyr-appen integrerar inte med några tredjepartstjänster för analys, annonsering eller spårning. De enda externa förfrågningarna går till offentliga status-API:er och Googles favikontjänst.

Barns integritet

Vultyr-appen samlar inte in data från någon, inklusive barn under 13 år. Webbplatsen loggar endast anonymiserade, aggregerade besöksräkningar.

Ändringar

Om denna policy ändras uppdaterar vi datumet ovan.
