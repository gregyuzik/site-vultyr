Política de privacidad

Última actualización: 11 de abril de 2026

Resumen

La app Vultyr no recopila, almacena ni transmite datos personales. El sitio web Vultyr (vultyr.app) usa Google Analytics sin cookies para comprender el tráfico agregado de visitantes. Esta página explica ambos puntos en detalle.

App: recopilación de datos

La app vultyr no recopila ninguna información personal. No requiere una cuenta, no incluye SDK de análisis o seguimiento de terceros y no envía datos a ningún servidor operado por nosotros.

App: solicitudes de red

La app realiza solicitudes HTTPS directas a API públicas de páginas de estado (como Statuspage.io, Apple, Google Cloud y otras) para comprobar el estado de los servicios. Estas solicitudes van directamente desde tu dispositivo a la API pública del servicio, no pasan por ningún servidor operado por nosotros.

App: almacenamiento de datos

Todos los datos se almacenan localmente en tu dispositivo usando el framework SwiftData de Apple. Si activas la sincronización con iCloud, tu lista de servicios monitoreados se sincroniza mediante iCloud Key-Value Store de Apple, regido por la política de privacidad de Apple. Nosotros nunca vemos estos datos.

App: alertas entre dispositivos

Si activas las alertas entre dispositivos, los cambios de estado se comparten entre tus dispositivos mediante iCloud Key-Value Store de Apple. Cuando tu Mac detecta un cambio de estado, escribe una señal ligera en tu cuenta de iCloud. Tus otros dispositivos observan el cambio y ejecutan su propia comprobación local. No interviene ningún servidor de terceros, toda la comunicación pasa por la infraestructura iCloud de Apple. Puedes activar o desactivar esta función desde cualquier dispositivo.

App: favicons

Los favicons de los servicios se obtienen del servicio público de favicons de Google y se guardan en caché localmente en tu dispositivo.

Sitio web: analíticas

Este sitio web (vultyr.app) usa Google Analytics 4 en modo sin cookies y con IP anonimizada para contar vistas de página agregadas. Concretamente, configuramos gtag.js con anonymize_ip: true, client_storage: 'none', allow_google_signals: false y allow_ad_personalization_signals: false. Esto significa que no se establece ninguna cookie _ga, tu IP se trunca antes de almacenarse y no se recopilan identificadores publicitarios. La propia app vultyr no incluye ningún tipo de analítica.

Sitio web: dominios de terceros

Al cargar vultyr.app se contactará con los siguientes dominios de terceros:

  • www.googletagmanager.com, carga el script gtag.js
  • www.google-analytics.com / *.analytics.google.com, recibe beacons anonimizados de vistas de página
  • www.google.com/g/collect, recibe los mismos beacons anonimizados de vistas de página (endpoint de recopilación de reserva de Google Analytics 4)

No cargamos Google Fonts (la fuente Audiowide está autoalojada en vultyr.app) y no utilizamos un servicio de favicons de terceros para las imágenes propias del sitio web.

App: servicios de terceros

La app vultyr no se integra con ningún servicio de análisis, publicidad o seguimiento de terceros. Las únicas solicitudes externas son a API públicas de estado y al servicio de favicons de Google.

Privacidad de menores

La app vultyr no recopila datos de nadie, incluyendo menores de 13 años. El sitio web solo registra recuentos de visitantes anonimizados y agregados.

Cambios
