Политика конфиденциальности

Обновлено: 11 апреля 2026 г.

Краткое содержание

Приложение Vultyr не собирает, не хранит и не передаёт никаких персональных данных. Сайт Vultyr (vultyr.app) использует Google Analytics без файлов cookie для понимания агрегированной статистики посещений. Ниже подробности по обоим пунктам.

Приложение — сбор данных

Приложение vultyr не собирает никакой персональной информации. Оно не требует регистрации, не содержит сторонних SDK для аналитики или трекинга и не отправляет данные на наши серверы.

Приложение — сетевые запросы

Приложение делает прямые HTTPS-запросы к публичным API статус-страниц (таким как Statuspage.io, Apple, Google Cloud и другим), чтобы проверять статус сервисов. Эти запросы идут напрямую с вашего устройства к публичному API сервиса — они не проходят через наши серверы.

Приложение — хранение данных

Все данные хранятся локально на вашем устройстве с использованием SwiftData от Apple. Если вы включите синхронизацию iCloud, список отслеживаемых сервисов синхронизируется через iCloud Key-Value Store от Apple, регулируемый политикой конфиденциальности Apple. Мы никогда не видим эти данные.

Приложение — уведомления между устройствами

Если вы включите уведомления между устройствами, изменения статуса передаются между вашими устройствами через iCloud Key-Value Store от Apple. Когда Mac обнаруживает изменение статуса, он записывает лёгкий сигнал в ваш аккаунт iCloud. Другие устройства замечают изменение и выполняют свою локальную проверку. Сторонние серверы не задействованы — вся коммуникация идёт через инфраструктуру iCloud от Apple. Вы можете включать и отключать эту функцию с любого устройства.

Приложение — фавиконы

Фавиконы сервисов загружаются из публичной службы фавиконов Google и кэшируются локально на вашем устройстве.

Сайт — аналитика

Этот сайт (vultyr.app) использует Google Analytics 4 в режиме без файлов cookie с анонимизацией IP для подсчёта агрегированных просмотров страниц. Конкретно мы настраиваем gtag.js со следующими параметрами: anonymize_ip: true, client_storage: 'none', allow_google_signals: false, allow_ad_personalization_signals: false. Это означает, что cookie _ga не устанавливается, ваш IP усекается до сохранения, рекламные идентификаторы не собираются. Само приложение vultyr никакой аналитики не содержит.

Сайт — сторонние домены

Загрузка vultyr.app связывается со следующими сторонними доменами:

  • www.googletagmanager.com — загружает скрипт gtag.js
  • www.google-analytics.com / *.analytics.google.com — принимает анонимизированные сигналы о просмотрах
  • www.google.com/g/collect — принимает те же анонимизированные сигналы (резервный эндпоинт сбора данных Google Analytics 4)

Мы не загружаем Google Fonts (шрифт Audiowide размещён на vultyr.app) и не используем сторонний сервис фавиконов для собственных изображений сайта.

Приложение — сторонние сервисы

Приложение vultyr не интегрируется ни с какими сторонними сервисами аналитики, рекламы или трекинга. Единственные внешние запросы — к публичным API статус-страниц и к службе фавиконов Google.

Конфиденциальность детей

Приложение vultyr ни у кого не собирает данные, в том числе у детей младше 13 лет. Сайт фиксирует только анонимизированные агрегированные счётчики посещений.

Изменения

Если эта политика изменится, мы обновим дату выше.
